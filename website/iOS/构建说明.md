# iOS 构建说明

本项目使用 Capacitor 将 Web 应用封装为 iOS App。

## 构建要求
- macOS 系统
- Xcode 15+
- Node.js 18+
- CocoaPods

## 构建步骤

1. 安装依赖：
   ```bash
   cd ios/App
   npm install @capacitor/core @capacitor/cli @capacitor/ios
   ```

2. 安装 CocoaPods 依赖：
   ```bash
   cd ios/App
   pod install
   ```

3. 打开 Xcode 项目：
   ```bash
   open App.xcworkspace
   ```

4. 在 Xcode 中：
   - 选择目标设备或 "Any iOS Device"
   - 配置签名（Signing & Capabilities）
   - Product > Archive 生成 IPA

## 注意事项
- 本项目 Bundle ID 为：com.sunxiaozhi.fanclub
- 需要在 Apple Developer 账号中配置相应的 App ID 和证书
